# ✨ Features

### 🎨 _Full Color Customization_
– Pick any color using an intuitive color menu with an adjustable **_Alpha_** slider to increase or decrease color intensity as you see fit.

### 🌈 _Rainbow Mode_
– A fun mode where all omens have their colors randomized by cycling through the color spectrum, along with a configurable **_Speed_** slider to control how fast or slow the colors change.

### ⚪ _White Mode_
– Another fun mode where all omens are turned into pure white. _(NOTE: This may cause rendering issues – use at your own risk)_

### 🎯 _Duty Overrides_
– Set custom colors for specific duties/instances. These override your global color settings, giving you per‑duty control.

### 🛡️ _Include Non‑Hostile Omens_
– Customize the colors of safe (non‑hostile) AoE omens/indicators as well, along with a separate color menu.

### 🧪 _Testing_
– Test all omens yourself and fine‑tune everything to your liking before using them in live content.

# 📦 Installation

### Step **_1_**: _Prerequisites_
- **[XIVLauncher](https://goatcorp.github.io/)** is the only requirement to use this plugin.

### Step **_2_**: _Add Repository_
1. Open the **_Dalamud Settings_** menu in‑game. You can do this by:
   - Clicking the button at the bottom of the Plugin Installer
   - Typing `/xlsettings` in chat
2. Navigate to the **_Custom Plugin Repositories_** section.
3. Paste the following repository URL into the input field:  
   - `https://puni.sh/api/repository/akechi`
4. Click the **_`+`_** button.
5. Click **_Save and Close_**.

### Step **_3_**: _Install Plugin_
1. Open the **_Dalamud Plugin Installer_** (type _`/xlplugins`_ in chat).
2. Select the **_All Plugins_** tab.
3. Search for **_Chroma_**.
4. Click **_Install_**.

### ✅ **_Done!_**

Chroma is now installed and ready to use.<br>
Feel free to get started by accessing the settings menu via the `/chroma` command.<br>
